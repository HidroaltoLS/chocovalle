# ChocoValle

Sitio web estático para la asociación de mujeres chocolateras de La Mana. Presenta su historia, proceso artesanal, catálogo de productos y canales de contacto en cuatro páginas diferenciadas.

## Contenido
- `index.html`: página de inicio con visión general, valores y colecciones destacadas para la venta.
- `nosotros.html`: historia de la colectiva, proceso artesanal y perfiles de las socias chocolateras.
- `productos.html`: catálogo de tabletas, bombones y ediciones especiales listas para pedidos.
- `contacto.html`: canales directos para realizar pedidos y seguir las novedades.
- `styles.css`: hoja de estilos compartida con tipografía elegante inspirada en el cacao.

## Uso
Abre cualquiera de los archivos `.html` en tu navegador preferido para explorar cada sección del sitio.
