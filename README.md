# ChocoValle

Sitio web estático para la asociación de mujeres chocolateras de La Mana. Incluye una página principal con información sobre la historia del colectivo, sus productos, procesos de elaboración, labor comunitaria y datos de contacto.

## Contenido
- `index.html`: estructura principal del sitio.
- `styles.css`: estilos principales inspirados en tonos de cacao.

## Uso
Abre `index.html` en tu navegador preferido para explorar el sitio.
