# ChocoValle

Sitio web estático para la asociación de mujeres chocolateras de La Mana. Presenta su historia, proceso artesanal, catálogo de productos y canales de contacto en cuatro páginas diferenciadas.

## Contenido
- `index.html`: página de inicio con enfoque comercial y destacados del catálogo.
- `nosotros.html`: historia de ASOLAMON, proceso artesanal y organización de los comités de trabajo.
- `productos.html`: catálogo detallado de bombones, tabletas y cajas de regalo con precios y especificaciones.
- `contacto.html`: canales directos (WhatsApp, correo y redes sociales) para coordinar pedidos.
- `styles.css`: hoja de estilos compartida con tipografía elegante inspirada en el cacao.

## Uso
Abre cualquiera de los archivos `.html` en tu navegador preferido para explorar cada sección del sitio.
